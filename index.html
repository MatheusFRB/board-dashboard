⚙️ Configuração Inicial — Microsoft Graph API

Para conectar o dashboard ao seu Excel Online, siga os passos abaixo uma única vez. Depois disso, os dados serão carregados automaticamente.

  1. Acesse o portal.azure.com → Azure Active Directory → App registrations → New registration. Nome: Board Dashboard, tipo: Single-page application, redirect URI: a URL do seu GitHub Pages.
  2. Após criar, copie o Application (client) ID e o Directory (tenant) ID.
  3. Vá em API permissions → Add a permission → Microsoft Graph → Delegated → adicione Files.Read e Sites.Read.All. Clique em Grant admin consent.
  4. Abra seu Excel no OneDrive / SharePoint. A URL terá um padrão como ...sharepoint.com/sites/SEU_SITE/... — copie o nome do site e o caminho do arquivo.
  5. Preencha os campos abaixo e clique em Salvar e Conectar.
